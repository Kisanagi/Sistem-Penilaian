# Sistem Pengolahan Nilai Akademik

Aplikasi web untuk mengelola nilai akademik siswa secara digital. Dirancang untuk tiga jenis pengguna: admin, guru, dan siswa.

## Fitur

### Admin
- Mengelola data guru dan siswa (tambah, edit, hapus)
- Mengelola mata pelajaran dan penugasan guru per kelas
- Melihat laporan rekap nilai seluruh siswa
- Mengekspor laporan nilai ke PDF

### Guru
- Menginput nilai siswa (tugas, UTS, UAS)
- Nilai akhir dan status kelulusan dihitung otomatis
- Melihat rekap nilai kelas yang diajar

### Siswa
- Melihat nilai per mata pelajaran
- Mengekspor laporan nilai pribadi ke PDF
